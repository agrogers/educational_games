# Hello World JS

This project contains a small interactive Binary Adding Quiz.

Quick start
- Open `index.html` with Live Server in VS Code (right-click the file → Open with Live Server).
- Or run a simple server from the project folder:

```powershell
npx live-server .
# or
python -m http.server 5500
```

How to use
- Choose the display format: `Binary` or `Decimal`. The question will be shown in that format and you must enter the answer in the same format.
- Choose how many numbers appear in each question (2–4) and a difficulty level (Easy/Medium/Hard).
- Press `Start` to begin a 10-question round. Submit answers, click `Next` to proceed, or `Show Answer` to reveal the correct result.

Notes
- The round consists of 10 autogenerated questions. `New Round` starts a fresh set. `Reset` clears progress.
- Levels control the maximum value per number (Easy=4 bits, Medium=8 bits, Hard=12 bits).

Files
- `index.html` — UI and controls.
- `script.js` — quiz logic.

If you want additional features (timer, persistence, different question counts, or "dismal" arithmetic), tell me which and I'll add them.
